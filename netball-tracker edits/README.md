# Netball Stats Tracker

A Progressive Web App (PWA) for tracking netball game statistics in real-time.

## Features
- Real-time stat tracking
- Game setup and roster management
- Post-game reports and CSV export
- Installable as a native app on mobile and desktop
- Works offline

## Installation
1. Host the files on a web server (HTTPS required for PWA features)
2. Open the app in a modern browser
3. Click "Install App" to add to your device

## Files
- `index.html` - Marketing website
- `app.html` - Tracker application
- `js/website.js` - Marketing website logic
- `js/app.js` - Tracker gameplay, saving, stats, and PWA logic
- `css/website.css` - Marketing website styling
- `css/app.css` - Tracker application styling
- `manifest.json` - PWA manifest
- `service-worker.js` - Offline caching
- `offline.html` - Offline fallback page
- `assets/icons/icon-192.png`, `assets/icons/icon-512.png` - App icons
- `assets/images/background.jpg.png` - Background image
- `assets/logos/Logo.jpg.png` - App logo

## Deployment
Upload all files to a web server. For GitHub Pages, push to a repo and enable Pages in settings.
